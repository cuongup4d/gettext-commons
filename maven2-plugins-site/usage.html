Usage

This plugin is used to generate Gettext PO files.

How to use the plugin

Update pot and po files: 

mvn org.xnap.commons:maven-gettext-plugin:merge

Create properties/classs files: 

mvn org.xnap.commons:maven-gettext-plugin:1.0:dist

How to Generate the Gettext Report

  <pluginRepositories>
    <pluginRepository>
      <id>xnap-commons</id>
      <name>XNap Commons Repository</name>
      <url>http://xnap-commons.sourceforge.net/repository</url>
    </pluginRepository>
  </pluginRepositories>

...

  <reporting>
    <plugins>
      <plugin>
        <groupId>org.xnap.commons</groupId>
        <artifactId>maven-gettext-plugin</artifactId>
        <version>1.1</version>
        <configuration>
          <targetBundle>my.package.Messages</targetBundle>
          <outputFormat>properties</outputFormat>
        </configuration>
      </plugin>
    </plugins>
  </reporting>
